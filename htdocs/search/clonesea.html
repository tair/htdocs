Using the Clone Search

The TAIR Clone Search window provides three ways of searching for a clone: simple search by name only, complex search using more limits, and search by position. These are offered in three distinct sections of the window. Each section has search and reset buttons that are active for the current section and all sections above them. So, for example, when you select the search in the middle (Add Constraints) section, the search is executed using all parameters you entered/selected in that section and in the Search by Name section above it.

Search by Name

At the top of the TAIR Clone Search window are the Search by Name options.

This simple search is case insensitive and allows wildcard searching (see "Using Wildcards").

Clone Name

Arabidopsis clones are usually named with the acronym of the vector followed by the plate and row numbers of the isolated clone. For example, CIC (YAC), T (TAMU BAC), F (IGF BAC) are some common vector acronyms.

Example clone names: CIC9A4, F16F4

Other Features

Attributes that limit the search. Check boxes let you select one or more attributes. Selecting multiple attributes indicates an "AND" relationship. For example, checking "is CDNA" and "is clone-end" limits the query set to cDNA clones and clone ends. A major type of these entries is the EST clones.

Further Restrict by Map Location

The bottom section of the window lets you restrict your search by location.

The options in this section let you use three parameters to restrict your search: Map Type, Chromosome, and Range.

Map Type

Lets you search entities by their position on a particular map and is to be used with the Range parameters. Currently, you can search on only one map type at a time. To search on multiple map types, use the TAIR Map Viewer.

Chromosome

Lets you limit your search to a single chromosome. There are five chromosomes in Arabidopsis: 1, 2, 3, 4, and 5.

Range

Lets you specify a range search by the upper and lower bounds (when you select "Between") or a center point (when you select "Around"). The value is interpreted based on the selected range units. You can specify the range by genetic distance (cM), physical distance (kb), and other entities on maps such as clone, marker, or gene names. When you select "Between" from the drop-down menu, your search will be within the range defined by two entities or positions on a particular map. When you select "Around" from the drop-down menu, your search will be the area +/-5 cM and/or +/- 200 kb from the specified entity or position. When you search around, the second value input and units options are disabled.
